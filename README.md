# i课表
i课表是一款开源，极简，适用于正方教务管理系统的课程表。
# 构思
整个App，计划只有两个界面，一个是课程表，一个是成绩表。无任何其他多余元素，致力于简约的风格
# 适用学校
理论上适合所有使用`正方教务管理系统`的大学
目前在以下大学测试通过
- 湖南工业大学
- 江苏理工学院

# 开发进度
- 用户登录
- 课程数据获取
- 课程表显示
- 成绩数据获取(未完成)
- 成绩表(未完成)
